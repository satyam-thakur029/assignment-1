The Central Processing Unit (CPU) is the brains of the computer. It is a microprocessor that executes instructions and processes data. It is responsible for carrying out the instructions of a computer program. It is the main component of a computer system and performs all the functions necessary for a computer to operate.

Data is stored in the computer's memory. This can be either RAM (Random Access Memory) or a hard drive. RAM is a type of volatile memory that stores data temporarily while a computer is running. Hard drives are non-volatile memory that stores data permanently.
